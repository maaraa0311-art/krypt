# KRYPT — Веб хөгжүүлэлтийн студи

Жижиг бизнес, стартап, хувийн брэндэд зориулсан веб хөгжүүлэлтийн студийн танилцуулга сайт.

## Бүтэц

- `index.html` — бүх зүйл нэг файлд (HTML + CSS + JS). Build шаардлагагүй.
- `krypt-logo.png` — брэндийн лого (nav болон footer дээр гарна).

## Ажиллуулах

`index.html`-г шууд хөтөч дээр нээх, эсвэл статик хостинг (Vercel, Netlify, GitHub Pages) дээр байршуулна.

```bash
# Жишээ: локал сервер
npx serve .
```

## Хэсгүүд

Hero · Үйлчилгээ · Чанарын стандарт (bento) · Үйл явц · Үнэ · Асуулт · Холбоо барих

## Лого

`krypt-logo.png` файл байхгүй бол nav дээр "KRYPT" текст автоматаар орлоно. Логогоо энэ нэрээр хадгалбал шууд харагдана.
